### 유형

짧은 운영글 / 생활형 빌드인퍼블릭 / 캐릭터형 운영

### 목적

- 계정의 분위기 유지
- 운영자 캐릭터 형성
- 사용자와 정서적 친밀감 형성
- 서비스 존재를 자연스럽게 노출
- “앱 운영 = 삶의 일부”처럼 보이게 만들기

### 전체적인 경향성

- 서비스 자체보다 “운영자의 생활감”이 먼저 보임
- 홍보보다 일기와 혼잣말에 가까움
- 앱을 스타트업처럼 포장하지 않음
- “성장”보다 “꾸준함”을 강조
- 거창한 성공서사 대신 소소한 지속성을 보여줌
- 독서앱 계정이라기보다 “책 좋아하는 사람 계정”처럼 운영됨

### 반복적으로 등장하는 패턴

### 1. 백수/게으름/현실감 캐릭터

- “현)백수”
- “방금 일어남;;;”
- “쉴까 말까 고민중”

→ 완벽한 창업자 이미지 제거

→ 친근감 생성

---

### 2. 독서 자체를 삶처럼 보여줌

- 야독
- 오디오북
- 현재 읽는 책
- 책 문장 인용

→ 앱 홍보보다 독서 생활 공유가 우선

---

### 3. 짧은 혼잣말 문체

- “정상인가요?”
- “왜그랬을까?”
- “통할지는 모르겠다”

→ 댓글 유도보다 공감 유도

---

### 4. 과장 없는 운영일지

- “90일차”
- “3개월차”

→ 조용한 지속성 강조

→ Build in Public의 저자극 버전

---

### 5. 서비스 홍보를 생활 속에 섞음

- 책장뷰 예고
- 독서모임
- 앱 운영일지

→ 광고처럼 안 보임

### 따라할 포인트

- 짧게 자주 올림
- 기능 설명보다 감정/생활 먼저
- 운영자를 캐릭터처럼 노출
- 실패/게으름/불안도 숨기지 않음
- 서비스와 일상을 분리하지 않음

### 우리 서비스 적용

- “운영자 캐릭터 기반 SNS 운영” 템플릿
- 저자극 Build in Public 스타일
- 광고 피로감 낮은 장기 운영 방식
