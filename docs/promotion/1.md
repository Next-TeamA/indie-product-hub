# **게시물 유형 DB 예시**

| **유형** | **목적** | **입력 데이터** | **출력 스타일** |
| --- | --- | --- | --- |
| 기능 출시 | 기능 알리기 | 기능명, 문제 | “드디어 ~ 추가” |
| 개발 회고 | 공감/스토리 | 오늘 한 일 | 회고형 |
| 문제 해결 | 전문성 | 해결한 버그 | 기술 공유형 |
| 성장 공유 | 신뢰 | 유저 수, MRR | 성과형 |

# **🚀 이게 엄청 중요함**

왜냐면 나중에 AI 프롬프트가:

톤:
- 인디해커 스타일

처럼 구조화 가능해짐.
